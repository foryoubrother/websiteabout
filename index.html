SP pneumatic 550

Принтер трафаретной печати с пневмоприжимом для нанесения паяльной пасты или клея на печатные платы для поверхностного монтажа. Устройство представляет собой полуавтоматический принтер для равномерного и точного нанесения паяльной пасты, в комплекте с которым поставляется устройство натяжения с возможностью регулирования степени натяжки по осям Х и Y. Оснащено пневмоприжимами ракеля к трафарету

рамы для натяжения трафаретов

Универсальные рамы для натяжения трафаретов позволяют обеспечить высокую эффективность и гибкость производства, снизить площадь хранения трафаретов, а также обеспечить значительную экономию средств предприятия за счет отказа от трафаретов на собственной раме.


Вакуумная печь оплавления припоя

Вакуумная инфракрасная печь для пайки печатных плат различной сложности габаритами до 300х300 мм. Каждая печатная плата будет спаяна с наивысшим качеством благодаря: Снижению количества пустот в паяном соединении до минимума; Равномерности нагрева печатной платы по площади в пределах 5°С; Отсутствию ошибок оператора.

Устройство для запрессовки разъемов SMTpress

Универсальная ручная оснастка для запрессовки PRESS-FIT разъемов на печатную плату с возможностью контроля усилия запрессовки. Пресс реечный для создания требуемого усилия запрессовки Держатель платы, настраиваемый по ширине, стопоры для фиксации ПП. Оснастка (матрица и пуансон) под 1 компонент Весовой индикатор для измерения и отображения усилия

Манипулятор вакуумный линейный с возможностью вертикального перемещения образца

КБ НП изготавливает линейные магнитные манипуляторы, обеспечивающие ввод поступательного и вращательного движения в вакуум, с возможностью установки механизма лифта подложкодержателя.Линейное перемещение и вращение штока манипулятора осуществляется с помощью наружного постоянного магнита  

Вакуумный откачной пост

Высоковакуумный пост необходим для создания высокого вакуума (1.0х10-6 мм.рт.ст) в технологических процессах хлорирования
